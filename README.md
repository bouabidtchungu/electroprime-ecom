# ElectroPrime E-commerce Platform

A professional, high-performance e-commerce platform for electronic devices. Built with **React**, **Tailwind CSS**, and **Express/Node.js**.

## Features

- **Dynamic CMS**: Edit Home Page, About Page, Footer, and Logo directly from the Admin Dashboard.
- **Product Management**: Add, Edit, and Delete products with image uploads.
- **Secure Admin**: Protected dashboard and API endpoints.
- **Responsive Design**: Optimized for mobile and desktop.

## Local Setup

1. Clone the repository.
2. Install dependencies:

   ```bash
   npm install --legacy-peer-deps
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Access the site at `http://localhost:3001`.

## Deployment

This project is ready for deployment on **Render.com** or **Railway.app**. See `deployment_guide.md` for details.

## Security

Secure your production environment by setting the `ADMIN_PASSWORD` environment variable.
